README
Material for TTB-ABC paper including all examples, figures, and text. 
Copyright@Schulz, Speekenbrink & Meder. 
Growing Heuristics via Approximate Bayesian Computation.

---------------------------------------------------------------------------------------------------------------
R: folder containing R-code

->BruteForceCity.R: brute force cue search in R for city data
->TradeOffTime.R: simulation of comutational time in dependency of rho, eta, and epsilon
->TradeOffPerformance.R: simulation of performance in dependency of rho, eta, and epsilon
->TreeDensity.R: plot of tree denisties after n samples
->LinearABCExample.R: Toy example of ABC in a linear set-up
->ttbabcfit.R: function to fit the TTB-ABC model
->ttbabcpredict.R: function to predict new data given a model fit by TTB-ABC
->CityCompare.R: code to compare standard TTB, TTB-ABC, and Classification Trees within the city size data set
->CityTTB.R: code to show that TTB-ABC generates sensible inferences within the city size data set
->RecoverTTB.R: code to show that TTB-ABC generates sensible inferences when the generating mechanism is TTB
---------------------------------------------------------------------------------------------------------------

figs: folder containing Plots as .pdf-files

->citytrace.pdf: plot of inferences within the city size data set
->linexample.pdf: plot for linear ABC example
->performancetradeoff.pdf: performance trade-off wireframe
->samplingscheme.pdf: figure showing how TTB-ABC's sampling works
->timetradeoff.pdf: time trade-off wireframe
->treedensity.pdf: plot of tree density for different numbers of observations
->ttbrecovertrace.pdf: plot of inferences within an environment that was generated by TTb
---------------------------------------------------------------------------------------------------------------

data: folder containing data files as .csv-files

->bruteforcecity.csv: result of the brute force comparison
->city.csv: original city size data set
->citycompare.csv: output of CityCompare.R, data set with performance of TTB, TTB-ABC, and Classification Trees
->performancetradeoff.csv: file containing the performance trade-off for different epsilon, rho, eta-combinations
->timetradeoff.csv: file containing the time trade-off for different epsilon, rho, eta-combinations


---------------------------------------------------------------------------------------------------------------

algs: folder containing the algorithms in tex's algorithmic format:

->abc.alg: Approximate Bayesian Computation
->ttbabc.alg: function for learning and predictions of TTB-ABC
